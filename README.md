# Warfame Tool

A web app used to check prices of in-game items.


## Installation

Install WarframeTool with npm

```bash
  cd WarframeTool
  npm install
```
    
## Deployment

To deploy this project run

```bash
  npm start server
```


## Features

- Daily updated databse
- Live previews
- Items details
- Advanced stats & graphs features
